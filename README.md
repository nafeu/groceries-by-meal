# Groceries By Meal

Decide your groceries by meal.

![Groceries By Meal By Nafeu](https://nafeu.com/groceries-by-meal/app-preview.png)

## Installation

```
git clone https://github.com/nafeu/groceries-by-meal.git
cd groceries-by-meal
npm install
```

## Development

```
npm start
```

## Deployment

```
npm run deploy
```
